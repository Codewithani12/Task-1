# Pet-FoodStore 🐾

A simple and responsive online pet-food shop built using HTML, CSS & JavaScript—featuring product browsing, add-to-cart functionality and checkout (all handled in-browser).

## ⚙️ Tech Stack  
HTML • CSS • JavaScript (Vanilla)

## 💡 Highlights  
- Responsive design: works across mobile, tablet & desktop  
- Product catalog display + “add to cart” logic in-browser  
- Checkout workflow simulated via client-side JavaScript  
- Clean, beginner-friendly front-end project—great for learning UI/UX and JavaScript basics  
- Fully self-contained: no backend required

## 🛎️ Usage  
Simply open `index.html` in your browser and explore the app — no server setup needed.
